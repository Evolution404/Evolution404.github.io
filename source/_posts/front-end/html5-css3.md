# html5
## 新标签

### 显示列表 details以及summary标签
```html
<details>
<summary>显示列表</summary>
<ul>
    <li>aaaaa</li>
    <li>bbbbb</li>
    <li>ccccc</li>
    <li>ddddd</li>
</ul>
</details>
```

### progress标签

```html
<progress min="0" max="100" value="50"></progress>
```
