## 使用Startify中自带的session功能

```
:SLoad       load a session    |startify-:SLoad|
:SSave[!]    save a session    |startify-:SSave|
:SDelete[!]  delete a session  |startify-:SDelete|
:SClose      close a session   |startify-:SClose|
```
